千行计划

本文为千行任务预览，同时也是一些基础的学习路线

  • Helloworld

    1. 使用printf输出”helloworld！”
    2. 使用iostream输出”helloworld！”

  • 数据类型

    1. 熟悉int，float，double，chat，string等几种常见数据类型

  • 逻辑结构

    1. 找出1~10000中13的倍数
    2. 输入一个分数，大于60输出”合格”，小于60输出”不合格”(使用switch)

  • 随机数

    1. 输出0~30之间的一个随机数

  • 类

    1. 写一个”狗”类
    2. 写一个”哈士奇”类，继承”狗”类
    3. 写一个”猫”类，继承”狗”类

  • Vector

    1. 使用数组存1~10000中13的倍数
    2. 使用vector存1~10000中13的倍数
    3. 使用vector存1~100，然后删除所有单数
    4. 使用vector存放任意数量的”狗”类，并尝试输出类的某一成员变量
    5. 使用vector存放分别存储1到100之间的单数、1到100之间的双数、1到100之间的质数的vector，并输出同时是单数和质数的数值

  • git

    1. 创建GitHub账号并创建一个公开仓库或私有仓库
    2. 安装git
    3. 本地建立仓库并连接到GitHub的仓库
    4. 尝试将自己的代码或一些文本通过git上传到GitHub的仓库
    5. 再建一个新的本地仓库，将自己在GitHub的仓库clone下来

  • 待续
