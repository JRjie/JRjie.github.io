Command_Line

Linux

文件操作

  • cd /
    • ==打开文件夹==
  • cd ..
    • ==返回上一级文件夹==
  • ls
    • ==查看当前路径文件==
  • rm -r //
    • ==删除文件夹==
  • rm -f
    • ==删除文件==
  • cp // //
    • ==复制文件&重命名==
  • mkdir
    • ==创建文件夹==
  • touch
    • ==创建文件==
  • tar -xvf
    • ==解压tar包==
  • unzip -d
    • ==解压zip包==
  • tar -cvf
    • ==压缩tar包==
  • zip -v
    • ==压缩zip包==

软件操作

  • apt-get update
    • ==更新软件源==
  • apt-get upgrade
    • ==更新全部软件==
  • apt-get upgrade
    • ==更新指定软件==
  • apt list –upgradable
    • ==可更新软件列表==
  • apt list –installed
    • ==查看已安装的所有软件==
  • apt-get install
    • ==安装软件==
  • apt-get remove
    • ==删除软件==
  • apt-get –purge remove
    • ==删除软件和配置文件==

ROS2

  • source /opt/ros/galactic/setup.bash

    • ==获取启动文件==

  • printenv | grep -i ROS

    • ==检查环境变量==

  • ros2 node list

    • ==查看节点列表==

  • ros2 topic list

    • ==查看话题列表==

  • ros2 param list

    • ==查看参数列表==

  • ros2 run

    • ==运行包的可执行文件==

  • ros2 node info

    • ==检查节点信息==

  • ros2 topic info

    • ==检查主题信息==

  • ros2 topic echo

    • ==查看节点数据==

  • ros2 interface show

    • ==查看主题期望的数据结构==

  • ros2 param get

    • ==查看参数的类型和当前值==

  • ros2 param set

    • ==修改参数当前值==

  • ros2 launch

    • ==打开启动文件==

  • ros2 launch

    • ==打开包中的启动文件==

  • ros2 bag record

    • ==记录话题信息==

  • ros2 bag record -o …

    • ==记录多个话题信息==

  • ros2 topic hz

    • ==查看话题数据发布频率==

  • colcon build

    • ==构建命名空间&构建包==

  • colcon build –package-select

    • ==构建某个包==

  • ros2 pkg create –build-type ament_cmake

    • ==创建新包==

Git

  • git config –global user.name ““
    • ==配置全局用户名==
  • git config –global user.email ““
    • ==配置全局邮箱==
  • ssh-keygen
    • ==获取公钥==
  • git init
    • ==初始化==
  • git clone
    • ==克隆仓库==
  • git add.
    • ==将全部文件放到暂存区==
  • git commit -m ““
    • ==将全部文件合并==
  • git remote add origin
    • ==连接远程仓库==
  • git push origin master
    • ==上传合并的文件==
  • git push origin master -f
    • ==强制上传==
  • git reset HEAD
    • ==将暂存区的所有文件恢复到工作区==
  • git remote rm origin
    • ==删除远程连接==
